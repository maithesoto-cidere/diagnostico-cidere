# Diagnóstico de Capacidades · CIDERE Biobío

Sistema de diagnóstico empresarial para el Programa Proveedores Locales.

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Sube este proyecto a GitHub
2. Entra a vercel.com → New Project → importa el repo
3. Vercel detecta Vite automáticamente → Deploy
